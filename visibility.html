Visibility of System Status:

The system should always keep users informed about what is going on, through appropriate feedback within reasonable time.

The selected menu choice is not highlighted. Instead all choices in the menu are highlighted. Sometimes the header matches the menu choice so that the status is visible, but this is not always the case. For example, the menu choice news has the headline communication and marketing.

The menu is however present at all times which makes it clear where the user can go next.
